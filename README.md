# go-file-undercursor README

## Tested Env

* Ubuntu 18.04

## Features

Vscode Extension for Go To File under cursor. Just like VIM 'gf' Function. 

Open file that follow the link under cursor



## Key Binding
Default Key bining is 

> `Ctrl + K` + `Ctrl + G`
